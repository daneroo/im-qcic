# Email notifications and templating

Using nodemailer, with mailgun transport, and these templates:

- [Premailer](http://premailer.dialect.ca/)
- [MUI-CSS boilerplate email](https://www.muicss.com/docs/v1/email/boilerplate-html)
- [CSS-TRicks Email workflow](https://css-tricks.com/tools-html-email-workflow/)
- [MailChimp templates](https://github.com/mailchimp/email-blueprints)
- [Zurb Foundation for Emails](https://foundation.zurb.com/emails.html)
- [Transactional email templates from mailgun](https://github.com/mailgun/transactional-email-templates)
- [Matreial Design HTML Template](https://www.behance.net/gallery/61655319/Free-HTML-Email-Template-Material-Design)
