## Fetch

```jsx
<Fetch url="https://time.qcic.n.imetrical.com/" 
  render={
  ({ loading, error, data })=>{
    if (loading) return <pre>Loading...</pre>
    if (error) return <pre>{error.name}: {error.message}</pre>
    return (<pre>
      {JSON.stringify(data)}
    </pre>)
  }
} />
```
