# Fetch

Uses RenderProps

```jsx
<Fetch url="https://time.qcic.n.imetrical.com/" render={
  ({ loading, error, data })=>{
    if (loading) return <p>Loading...</p>
    if (error) return <p>{error.name}: {error.message}</p>
    return (<pre>Tada:{JSON.stringify(data)}</pre>)
  }
} />
```
