# Demo

This shows is how you would use `<RegisterApolloProvider />` with a query. Similarly with a subscription.

```jsx
<RegisterApolloProvider httpurl='https://fizzbuzzclock.n.imetrical.com/graphql'>
  <Query query={helloQy} >
    {({ loading, error, data }) => {
      if (loading) return <p>Loading...</p>
      if (error) return <p>Error :(</p>
      return <div>Hello: <b>{data.hello}</b></div>
    }}
  </Query>
</RegisterApolloProvider>
```
