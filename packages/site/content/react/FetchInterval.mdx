# FetchInterval

```jsx
<FetchInterval url="https://time.qcic.n.imetrical.com/" delay={1000} />
```

<FetchInterval url="https://time.qcic.n.imetrical.com/" delay={1000} />

Uses RenderProps

```jsx
<FetchInterval url="https://time.qcic.n.imetrical.com/" delay={2000} render={
  ({ loading, error, data })=>{
    if (loading) return <pre>Loading...</pre>
    if (error) return <pre>{error.name}: {error.message}</pre>
    return (<pre>Tada: {JSON.stringify(data)}</pre>)
  }
} />
```
